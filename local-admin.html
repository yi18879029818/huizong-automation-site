Local Form Submission Admin

这是本地版后台页面。直接打开这个 HTML 文件，填入线上站点地址、管理员账号和密码，就能从 Cloudflare Pages 读取表单统计与最近提交记录。

Local Admin File

连接设置

默认站点已经预填为当前线上地址。如果以后换了站点域名，只需要改这里，不用再改代码。

怎么使用

  1. 双击打开这个 `local-admin.html` 文件。
  2. 确认 `Site URL` 是你的线上站点地址。
  3. 输入你在 Cloudflare Pages 里配置的 `ADMIN_USERNAME` 和 `ADMIN_PASSWORD`。
  4. 点击 `Connect Dashboard`，本地页会直接请求线上 `/api/admin/stats` 和 `/api/admin/submissions`。
