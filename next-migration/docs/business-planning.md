# Fleet Website 业务策划文档

## 1. 项目背景

`fleet.coolyne.com` 是面向工业自动化、仓储物流自动化、AGV/AMR 产品与解决方案展示的企业官网。

项目目标不是单纯做一个“展示站”，而是建设一个同时承担以下职责的营销型站点：

- 对外建立品牌可信度
- 清晰展示产品线与应用能力
- 引导潜在客户发起询盘
- 承接资料下载、快速报价、技术支持等转化动作
- 为后续 SEO、内容投放、销售线索沉淀提供基础设施

## 2. 业务目标

### 2.1 品牌目标

- 建立 `Fleet` 品牌统一形象
- 强化“工业自动化、仓储机器人、系统集成”的专业认知
- 通过案例、证书、产品能力页面提升可信度

### 2.2 获客目标

- 提升网站自然询盘量
- 为销售团队沉淀更高质量的项目线索
- 降低用户首次沟通门槛

### 2.3 转化目标

站点重点转化动作：

- 联系我们提交询盘
- 获取快速报价
- 预约现场勘察 / site visit
- 下载产品资料
- 提交技术支持需求

## 3. 目标用户

### 3.1 核心用户画像

- 制造业工厂负责人
- 仓储物流项目负责人
- 自动化设备采购人员
- 系统集成合作伙伴
- 海外渠道/代理客户

### 3.2 用户关注点

- 产品是否匹配实际工况
- 能否提供整线/整仓解决方案
- 是否具备交付能力与行业经验
- 是否有认证、专利、客户案例
- 报价和技术响应是否快速

## 4. 网站定位

网站定位为：

**“工业自动化与智能仓储解决方案官网 + 营销转化入口 + 项目线索收集平台”**

不是单纯资讯型网站，也不是单纯产品手册站。

## 5. 内容策略

### 5.1 一级内容结构

- Home：首页品牌与核心能力概览
- About：公司介绍、资质、证书、工厂形象
- Products：产品总览与产品详情
- Solutions：解决方案展示
- Case Studies：案例与行业应用
- Contact：询盘、报价、项目沟通入口

### 5.2 重点内容模块

- Hero 品牌主视觉
- 产品卡片与详情页
- 解决方案框架
- 合作品牌 / 客户背书
- 证书与荣誉
- FAQ / 技术咨询入口
- 聊天浮窗与下载入口

## 6. 转化设计思路

### 6.1 主转化入口

- Header CTA：`Speak With An Expert`
- Contact 页面表单
- 右下角 `Chat with Us`

### 6.2 辅助转化入口

- 产品页内 CTA
- 快速下载产品资料
- 技术支持快捷入口
- 预约现场勘察

### 6.3 转化策略原则

- 降低首次发问难度
- 让用户随时能发起联系
- 不要求用户先理解全部产品再咨询
- 把“获取报价”和“技术咨询”拆开处理

## 7. 品牌视觉方向

当前品牌方向：

- 主色系：深蓝 + 橙色
- 视觉关键词：工业、可靠、科技、国际化
- 页面风格：结构化、专业、具备一定技术感

视觉优化原则：

- 避免过重、压迫感太强的纯深蓝大面积铺底
- 保持国际工业 B2B 网站的稳定感
- 在“可信”和“现代”之间保持平衡

## 8. SEO 与内容增长方向

### 8.1 SEO 重点

- 产品关键词
- 解决方案关键词
- 行业应用关键词
- 长尾问题内容

### 8.2 可扩展内容方向

- AGV/AMR 产品知识文章
- 仓储自动化选型指南
- 不同行业解决方案文章
- 证书/认证解读
- 项目案例深度文章

## 9. 后续业务建议

### 短期

- 完善产品资料下载链路
- 优化联系页面 intent 预填
- 建立询盘来源标记

### 中期

- 增加可持续内容发布机制
- 增加 FAQ / 技术知识内容
- 结合 SEO 做产品页扩展

### 长期

- 接入 CRM / 线索管理系统
- 建立多语言版本
- 为销售与渠道提供区域化落地页
